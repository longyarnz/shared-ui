# Shared-UI.

## Intro
A simple app displaying canvas design components for [fireflies.ai](https://fireflies.ai).  

## How To Use
Open the terminal in the project folder and run:
```sh
  npm install
```
or run yarn
```sh
  yarn
```
The dependies will be installed on your local machine.  
Then:
```sh
  npm start
```
or run yarn
```sh
  yarn start
```
